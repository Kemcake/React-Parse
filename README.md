React-Parse
===========

Sample JS App using ReactJS and Parse.com

# Getting Started
Three simple steps

* `git clone https://github.com/Kemcake/React-Parse.git`
* `npm install`
* `npm start` and your app is build on every change

# Built with 
What's in the box
* ReactJS
* Parse.com SDK
* Bootstrap
* Sass
* Browserify
